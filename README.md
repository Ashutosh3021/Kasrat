A minimalist, offline-first fitness tracker that actually works at the gym. No signup. No ads. No excuses.

## Why? 🤔

Because your gym has no signal and your notes app sucks. **Kasrat** (Hindi: कसरत — exercise) tracks your lifts, shows progress graphs, and runs rest timers—fully offline, forever free.

## Stack

| Tech | Why |
|------|-----|
| React 19 + TS | Fast UI, type-safe |
| Vite 7 | Builds in milliseconds |
| Tailwind CSS 4 | Dark theme, 0 effort |
| Dexie.js | IndexedDB that doesn't hurt |
| Zustand | State without the boilerplate |
| Recharts | Pretty graphs for your ego |

## Run it locally

```bash
npm install
npm run dev
```
